# Multi-sensor-Arduino-car-project-
ESP32 DisasterBot — A smart autonomous rescue/safety robot featuring: 
• Fire detection &amp; escape logic 
• Gas leak monitoring (MQ-135) 
• Ultrasonic obstacle avoidance 
• WebSocket remote control 
• Smart buzzer emergency alerts
• 4dc motor control with L298N 
• Real-time environmental sensing
